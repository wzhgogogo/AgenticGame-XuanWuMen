## 详细说明

### 1. 案台背景 `desk.png` ✅

- 当前已有，作为全屏 `background: cover` 使用
- 深色木质纹理，左下角含烛台+蜡烛
- 如需升级：可考虑更高分辨率版本（2560×1440）

### 2. 长安城坊图 `map.png` ✅

- 当前已有，作为 `background: contain` 居中显示
- **位置**：屏幕中央偏上，占据约 40% 宽度
- **风格**：仿古地图，绢帛/宣纸底色（#d9c088 ~ #b8956a）
- **内容要求**：
  - 太极宫（左上方，红色边框）
  - 大明宫（右上方，虚线框）
  - **玄武门（金色高亮，核心位置）**
  - 兴庆宫
  - 东市、西市
  - 朱雀门、延兴门
  - 外城墙轮廓 + 坊市网格
- **交互提示**：可行动地点用菱形标记，太子党势力用红色方块标记
- **装饰**：四角有老化斑点，右上角可加印章

### 3. 活动图标

12 个活动各需一个专属图标：

| 活动 | 图标描述 | 建议文件名 |
|------|---------|-----------|
| 治政 | 竹简/卷轴 | `icon-govern.png` |
| 军务 | 交叉兵器 | `icon-military.png` |
| 情报 | 眼睛/暗哨 | `icon-intel.png` |
| 社交 | 对坐人物 | `icon-social.png` |
| 个人 | 冥想/独坐 | `icon-personal.png` |
| 巡查 | 马匹 | `icon-patrol.png` |
| 招募 | 兵卒 | `icon-recruit.png` |
| 训练 | 操练兵器 | `icon-train.png` |
| 屯田 | 稻穗 | `icon-farm.png` |
| 工务 | 工具/锤 | `icon-craft.png` |
| 外事 | 信使/书信 | `icon-diplomacy.png` |
| 祭祀 | 鼎 | `icon-ritual.png` |

- **风格统一**：圆形浮雕，铜质底色（#6b5030 → #1a0f08 径向渐变），金色线条（#c9a84c）
- **尺寸**：52×52，@2x 出 104×104

### 4. 奏折卡片底纹 `memorial-bg.png`

- 泛黄旧纸纹理，带横向竹纹条纹
- 可平铺或拉伸使用
- 配色参考：#e8dcc0 ~ #d4c098

### 5-6. 印章

- 大印（56×56）：地图上可拖动的"秦王"印
- 小印（20×20）：奏折卡片角落装饰

### 7. 时段按钮底纹

- 底部"晨时/午时/夜时"切换按钮的底纹
- 石碑或竹简风格
- 激活态有微弱金色光晕

## 优先级排序

1. **P0（核心体验）**：长安城坊图、活动图标×12
2. **P1（氛围提升）**：印章（大+小）、奏折卡片底纹
3. **P2（锦上添花）**：时段按钮底纹

## 配色参考

- 主色调：深木色 `#2a1d12` / 金色 `#c9a84c` / 暖黄 `#e8d4a8`
- 强调色：朱红 `#c73e3a`（印章/紧急标签）
- 纸张色：`#e8dcc0` ~ `#d4c098`
- 文字色：`#3a2010`（地图标注）/ `#d4b878`（UI 标题）
