## 详细说明

### 1. 案台背景 `desk.png` ✅

- 当前已有，作为全屏 `background: cover` 使用
- 深色木质纹理，左下角含烛台+蜡烛
- 如需升级：可考虑更高分辨率版本（2560×1440）

### 2. 长安城坊图 `map.png` ✅

- 当前已有，作为 `background: contain` 居中显示
- **位置**：屏幕中央偏上，占据约 40% 宽度
- **风格**：仿古地图，绢帛/宣纸底色（#d9c088 ~ #b8956a）
- **内容要求**：
  - 太极宫（左上方，红色边框）
  - 大明宫（右上方，虚线框）
  - **玄武门（金色高亮，核心位置）**
  - 兴庆宫
  - 东市、西市
  - 朱雀门、延兴门
  - 外城墙轮廓 + 坊市网格
- **交互提示**：可行动地点用菱形标记，太子党势力用红色方块标记
- **装饰**：四角有老化斑点，右上角可加印章

### 3. 活动分类图标

5 个分类各需一个图标：

| 分类 | 图标描述 | 建议文件名 |
|------|---------|-----------|
| 治政 | 竹简/卷轴 | `icon-governance.png` |
| 军务 | 交叉兵器 | `icon-military.png` |
| 情报 | 眼睛/暗哨 | `icon-intelligence.png` |
| 社交 | 对坐人物 | `icon-social.png` |
| 个人 | 冥想/独坐 | `icon-personal.png` |

- **风格统一**：圆形浮雕，铜质底色（#6b5030 → #1a0f08 径向渐变），金色线条（#c9a84c）
- **尺寸**：52×52，@2x 出 104×104

### 4. 奏折卡片底纹 `memorial-bg.png`

- 泛黄旧纸纹理，带横向竹纹条纹
- 可平铺或拉伸使用
- 配色参考：#e8dcc0 ~ #d4c098

### 5-6. 印章

- 大印（56×56）：地图上可拖动的"秦王"印
- 小印（20×20）：奏折卡片角落装饰

### 7. 时段按钮底纹

- 底部"晨时/午时/夜时"切换按钮的底纹
- 石碑或竹简风格
- 激活态有微弱金色光晕

### 8. 快进场景动画/图片

- 快进推演（快进 3 日 / 快进 7 日）时的过渡视觉
- 当前快进期间仅显示文字 loading，缺少沉浸感
- **可选方案**：翻日历动画、沙漏/日晷转动、水墨晕染过渡、卷轴展开等
- 需覆盖全屏或中央区域，配合"推演中……正在快进 X 天"文案

### 9-15. 事件场景背景（7 张）

7 个通用场景背景覆盖全部 11 个事件骨架，复用率高：

| 场景 | 文件名 | 适用骨架 |

| 宫殿朝堂 | `scene-court.png` | imperialSummons, courtImpeachment, courtCounterstrike |
| 宴席 | `scene-banquet.png` | banquetCrisis |
| 府邸对峙 | `scene-confrontation.png` | politicalConfrontation, subordinateUltimatum |
| 暗巷伏击 | `scene-ambush.png` | assassinationCrisis |
| 军营校场 | `scene-military-camp.png` | militaryConflict, seizeMilitaryCommand |
| 玄武门 | `scene-xuanwumen.png` | 终局决战 |

- **风格统一**：写实水墨风，与 desk.png 同色系（深木色 #2a1d12 / 暖黄 #e8d4a8）
- **尺寸**：1920×1080，作为全屏 background: cover 使用
- **要求**：画面上半留空给对话 UI，下半为主体场景

### 16. 标题画面背景 `title-bg.png`

- 游戏启动第一屏，目前 App.tsx 用纯 CSS 渐变
- **风格**：大气磅礴，远景长安城轮廓或宫殿剪影，黎明/黄昏氛围
- 需留出中央区域放标题文字和"开始游戏"按钮

### 17. 结局画面背景 `ending-bg.png`

- 游戏结局展示页，目前 EndingScreen.tsx 用纯 CSS 渐变
- **风格**：水墨卷轴展开感，可通用于 5 种结局（文字区分），或按结局分 5 张
- 需留出中央大面积放结局文案
